Plates

p5.pave2riso supports four stroke types for rendering path outlines:

Solid: Basic solid line with uniform color and weight.

Dashed: Line with customizable dash patterns using the dashArray property.

Pattern: Line filled with p5.pattern textures like dots, stripes, or waves.

Gradient: Line with gradient transitions between color stops along the path.
