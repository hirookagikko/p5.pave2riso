Welcome to p5.pave2riso! This minimal example shows the basic workflow: import libraries, create Riso channels, use the p2r() factory to render a shape, and call drawRiso(). That's all you need to start creating Risograph-style graphics.

The p2r() factory binds your Riso channels and canvas size once, then you can call it repeatedly to add paths to those channels. Here we render a single red circle at the center of the canvas.
