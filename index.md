# p5.pave2riso

Pave.jsで作成したベクターパスを、p5.riso.jsの複数チャンネルに一括展開するライブラリ

---

## ドキュメント

[docs/](docs/) - セットアップ、Fill/Stroke、エフェクト、モードなどの解説

## サンプル

- [Getting Started](examples/getting-started.html) - 最初の一歩
- [Fill Types](examples/fill-types.html) - 塗りつぶしの種類
- [Stroke Types](examples/stroke-types.html) - 線の種類
- [Effects](examples/effects.html) - Filter / Halftone
- [Print Modes](examples/print-modes.html) - overprint / cutout / join
- [Text Paths](examples/text-paths.html) - テキストをパスに変換

---

## インストール

```bash
npm install p5.pave2riso
```
